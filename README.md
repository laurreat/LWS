# Learn With Sena 2.0

Plataforma interactiva de aprendizaje de inglés para niños (6-14 años).

## 🚀 Características

- 🎮 **8 juegos interactivos**: Vocabulary, Phrases, Grammar, Sentence, Listening, Speaking, Spelling, Memory
- 📊 **Sistema de progreso**: Puntos, rachas, logros y rankings
- 📱 **PWA**: Funciona offline en móviles y tablets
- 🌙 **Modo oscuro/claro**
- 🎨 **UI moderna** con animaciones suaves

## 🛠️ Tecnologías

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand (gestión de estado)
- Web Speech API

## 📦 Instalación

```bash
npm install
npm run dev
```

## 🚀 Deploy en Vercel

1. Conecta tu repositorio en Vercel
2. Deploy automático

## 📁 Estructura

```
src/
├── app/              # Páginas Next.js
│   ├── games/         # Juegos
│   ├── progress/      # Página de progreso
│   └── resources/     # Recursos educativos
├── components/        # Componentes React
│   ├── ui/           # Componentes base
│   ├── layout/       # Navbar, Footer
│   └── games/        # Componentes de juegos
├── hooks/            # Custom hooks
├── lib/              # Utilidades y store
├── data/             # Contenido educativo
└── types/            # Tipos TypeScript
```

## 🎮 Juegos

| # | Nombre | Descripción |
|---|--------|-------------|
| 1 | Vocabulary | Quiz de vocabulario con 3 niveles |
| 2 | Phrases | Frases completas en inglés |
| 3 | Grammar | Preguntas de gramática |
| 4 | Sentence | Organiza las palabras |
| 5 | Listening | Practica tu oído |
| 6 | Speaking | Habla en inglés |
| 7 | Spelling | Practica spelling |
| 8 | Memory | Encuentra las parejas |
